# 現場写真リアルタイム管理システム - 企画プラン

## 概要

現場調査員（管理者兼任）がスマホで撮影した写真を、事務所PCでリアルタイムに管理するWebシステム。

---

## 利用シーン

- **撮影者**: 現場の調査員（管理者兼任）、土曜日に1人で撮影、今後増員の可能性あり
- **撮影デバイス**: スマホ（モバイル回線）
- **管理PC**: 事務所に設置（社内LAN）
- **被写体**: 主に電柱およびそれに絡む風景
- **規模**: 1現場あたり最大50枚、1日最大5現場で約300枚/日

---

## ネットワーク構成

- スマホ: モバイル回線（4G/5G）
- PC: 社内ネットワーク（有線LAN or 無線LAN）
- **別ネットワークのため、インターネット上にサーバーを設置する構成**

---

## メタ情報

### 自動取得（入力不要）

- 撮影日時
- GPS座標（緯度・経度）
- 撮影方向（コンパス）

### 手動入力

- **ビル名**（エリアを表す。フォルダ分けの単位。1つの電柱が複数ビル名を持つ場合あり）
- **工事番号 / 工事名**
- **電柱番号**
- **コメント**（任意）

### JPEGファイルへのメタ情報埋め込み

- EXIF / XMP を使い、ビル名・電柱番号・工事番号等をJPEGファイル自体に書き込む
- システム外に持ち出しても情報が残るようにする
- タグ機能として実装

---

## フォルダ分類

- **ビル名ごと**にフォルダを分ける
- 理由: 工事が違っても同じエリアの写真を再利用する可能性があるため
- 工事名でくくるよりエリア（ビル名）でくくる方が実用的

---

## ビル名の入力方式

- **事前登録リストからの選択** + **自由入力（新規追加）** の両方に対応

---

## 図面機能

- 工事に関連するPDF図面や写真をアップロードし、いつでも閲覧できるようにする
- デスクトップやカメラ機能で取り込む
- 撮影ポイントのマッピング機能は現段階では不要

---

## PC管理画面の機能

- 写真一覧のリアルタイム表示（撮影したら即反映）
- フィルタリング
  - ビル名で絞り込み
  - 電柱番号で絞り込み
  - 工事番号で絞り込み
- 写真の検索
- クリップボードへのコピー
- 写真の削除・整理

---

## データの永続性

- 過去の写真はすべて残す（過去にさかのぼって参照する需要あり）
- バックアップ機能が必要

---

## 段階的開発プラン

### フェーズ1（最小構成・まずここから）

- スマホで写真撮影 → GPS座標を自動取得 → サーバーに送信
- PCのブラウザで写真一覧がリアルタイム表示される
- ビル名を入力して撮影、ビル名ごとにフォルダ分け
- ビル名は「リスト選択 + 自由入力」の両対応

### フェーズ2

- 電柱番号・工事番号・コメントの入力
- JPEGファイルへのメタ情報埋め込み（EXIF/XMP）
- フィルタ・検索機能
- 図面PDF閲覧

### フェーズ3

- バックアップ機能
- クリップボードコピー
- 1つの電柱に複数ビル名を持たせる対応
- 増員対応（複数ユーザー管理）

| **Render**              | 無料〜$7   | GitHubと連携してpushするだけでデプロイ。設定がとにかく簡単。無料枠はスリープあり | ★★★   |
| **Railway**             | $5〜     | モダンなUI、DB込みで簡単に構築可能。使った分だけ課金で分かりやすい           | ★★☆   |
| **Hetzner VPS**         | €4〜     | ヨーロッパ拠点だが格安のVPS。自由度は最も高いが、自分でサーバー管理が必要        | ★☆☆   |
| **Conoha VPS / さくらVPS** | ¥700〜   | 日本のサービス。日本語サポートあり。ただしサーバー管理は自力                | ★☆☆   |


### 写真ストレージ（大量の画像を保存する場所）


| サービス              | 月額目安               | 特徴                              |
| ----------------- | ------------------ | ------------------------------- |
| **Cloudflare R2** | 10GB無料、以降$0.015/GB | S3互換API。転送量が無料。写真配信に最適          |
| **AWS S3**        | $0.023/GB          | 業界標準。情報が多く学びやすい。ただし転送量に課金あり     |
| **Backblaze B2**  | 10GB無料、以降$0.005/GB | 最安クラス。Cloudflareと組み合わせると転送量も無料に |


### データベース（メタ情報・検索用）


| サービス            | 月額目安   | 特徴                                 |
| --------------- | ------ | ---------------------------------- |
| **Supabase**    | 無料〜$25 | PostgreSQL。無料枠が大きい。管理画面付き。APIも自動生成 |
| **PlanetScale** | 無料〜    | MySQL互換。ブランチ機能でDB変更が安全             |
| **Turso**       | 無料〜$9  | SQLite互換でシンプル。軽量アプリ向き              |


### 推奨構成


| 役割     | サービス              | 理由                                        |
| ------ | ----------------- | ----------------------------------------- |
| サーバー   | **Render**        | 最も簡単。GitHubにpushするだけでデプロイ完了。今後APIを増やすのも容易 |
| 写真保存   | **Cloudflare R2** | 転送量無料で写真配信に最適。S3互換なので将来AWS移行も容易           |
| データベース | **Supabase**      | 無料枠が大きく、PostgreSQLの知識は汎用的。管理画面も便利         |


**月額コスト目安:** 無料枠内〜月$10程度

### 手軽構成（サービス2つで完結）


| 役割        | サービス                           |
| --------- | ------------------------------ |
| サーバー      | **Render**                     |
| 写真保存 + DB | **Supabase**（ストレージ機能もあるので兼用可能） |
